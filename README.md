# SOC Analyst Home Lab

## Overview
This project demonstrates a hands-on SOC lab built to simulate real-world security operations including monitoring, detection, and vulnerability management.

## Lab Components
- Security Onion (SIEM, monitoring, threat hunting)
- pfSense (firewall and router)
- Kali Linux (penetration testing)
- Nessus (vulnerability assessment)
- Ubuntu Server (infrastructure)

## Key Capabilities
- Log monitoring and alert analysis
- Alert triage and investigation
- Threat hunting using SIEM
- Vulnerability scanning and risk identification
- Basic incident escalation workflows

## Example Activities
- Investigated alerts generated in Security Onion
- Analysed network traffic for suspicious behaviour
- Conducted Nessus scans to identify vulnerabilities
- Simulated attacker activity using Kali Linux

## Certifications
- CompTIA A+, Network+, Security+
- ISC2 CC
- Qualys VMDR
- Fortinet Cybersecurity Fundamentals & Associate

## Status
Actively developing and expanding lab with new detection and investigation scenarios.
